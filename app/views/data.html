Contributing data to the node

If you have a dataset which would be useful to the DEFRA Group, is available under Open Government Licence or could be licensed for use by the DEFRA Group, then please get in touch to discuss adding it to the portal. In order to add a dataset to the portal we will need the following:

  1. The dataset
  2. Dataset metadata (ideally in Gemini II format)
  3. Styling file for displaying the data ie layer file, stylesheet.
  4. Licensing covering the DEFRA Group and contractors, either under OGL or proprietary licence.

Also this note on licensing:
To note: Some of these datasets are available for use under the Open Government Licence, and some come with more stringent use restrictions. More information on data use will be made available soon through dataset metadata and the download facility.
